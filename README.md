# AI Coding Mentor

An AI-powered coding practice platform that helps students improve their programming skills through problem solving, progress tracking, and instant feedback.

---

## Project Overview

AI Coding Mentor is a web-based coding practice system built using Python and Flask.  
It allows students to solve coding problems, submit solutions, track their progress, and receive simple automated feedback.

This project demonstrates backend development, database handling, and frontend integration.

---

## Features

• Coding practice environment  
• Built-in code editor for solving problems  
• Code submission system  
• Instant feedback system  
• Coding progress dashboard  
• Submission history tracking  
• Coding journey visualization
