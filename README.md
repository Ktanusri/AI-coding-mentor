# 🚀 AI Coding Mentor

An interactive coding practice platform designed to help students improve problem-solving skills through structured practice, real-time feedback, and progress tracking.

---

## 🔍 Overview

AI Coding Mentor is a full-stack web application built using Python and Flask that enables users to practice coding problems, track their performance, and improve consistency.

It focuses on creating a structured learning experience with features like submission tracking, accuracy monitoring, and guided problem-solving.

---

## ✨ Key Features

* 🧠 Coding practice environment with built-in editor
* ⚡ Code submission and evaluation system
* 📊 Progress tracking with accuracy metrics
* 🔥 Streak system to encourage consistency
* 🗂 Problem filtering (difficulty & category)
* 📈 Submission history and performance insights

---

## 🛠 Tech Stack

**Backend:** Python, Flask
**Frontend:** HTML, CSS, JavaScript
**Database:** SQLite
**Tools:** Git, GitHub
